Context Relay is a Chrome extension that lets you define your project context once, generate a structured "primer", and inject it into any AI chat with a single click.

Privacy-first

4 STEPS

How It Works

  1. CREATE
    Define a project with task, decisions, links, notes, and tags
  2. GENERATE
    Click "Generate Primer" to produce a structured context document
  3. RELAY
    Go to any AI chat → click the floating Relay button → select project
  4. DONE
    The AI has full context. No re-explaining. Start building.
